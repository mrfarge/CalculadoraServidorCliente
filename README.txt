------Servidor y cliente HTTP calculadora ------

Implementación de un servidor que a partir de dos valores y un operador realiza la operación indicada por estos y un cliente que envía los valores y operadores que extrae de cada una de las filas de un csv y con un request para obtener los resultados de las operaciones y guardarlos posteriormente en un fichero txt o csv a elección.
